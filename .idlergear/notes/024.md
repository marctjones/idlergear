### CLI Commands
```bash
idlergear wiki push              # Push references to Wiki
idlergear wiki pull              # Pull Wiki to references
idlergear wiki sync              # Bidirectional sync
idlergear wiki sync --conflict local   # Prefer local
idlergear wiki sync --conflict remote  # Prefer remote
idlergear wiki status            # Show sync status
idlergear wiki config            # Configuration
```

### Conflict Resolution
- **Manual** (default): List conflicts for user review
- **Local**: Always use IdlerGear version
- **Remote**: Always use Wiki version
- No overwrites without permission!

### Configuration
```toml
[wiki]
enabled = true
auto_sync = true
sync_interval = 300  # seconds
```

## ‚úÖ Option B: Watch Mode

### Implementation
**File**: `src/idlergear/watch.py` (350+ LOC)

**Features**:
- File system monitoring via git status
- TODO/FIXME/HACK comment detection
- Uncommitted changes tracking
- Smart debouncing (configurable)

### Proactive Prompts
1. **"5 files changed, commit now?"** - Triggers at threshold
2. **"TODO detected ‚Üí create task?"** - New TODO comments
3. **"FIXME detected ‚Üí create bug?"** - New FIXME comments
4. **"HACK detected ‚Üí create tech debt?"** - New HACK comments
5. **"100+ lines changed"** - Large uncommitted changes

### CLI Commands
```bash
idlergear watch start            # Start watching
idlergear watch start -i 5       # Custom interval (5s)
idlergear watch status           # Show current stats
idlergear watch config           # View/edit configuration
idlergear watch config enabled true  # Enable watch mode
```

### Configuration
```toml
[watch]
enabled = false
debounce = 30  # seconds
thresholds.files_changed = 5
thresholds.uncommitted_lines = 100
thresholds.test_failures = 1
detect_todos = true
detect_fixmes = true
detect_hacks = true
```

### Smart Features
- **Debouncing**: Don't prompt too often (configurable)
- **Thresholds**: Configurable triggers
- **Multi-language**: Detects TODO/FIXME/HACK in Python, JS, TS, Java, C, C++, Rust, Go

### Impact
Solves **proactive knowledge capture** - automates manual tracking!
