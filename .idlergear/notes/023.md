**Implementation:**
- `src/idlergear/wiki.py` - Complete wiki sync module (400+ LOC)
- Bidirectional sync between IdlerGear references and GitHub Wiki

- CLI commands: push, pull, sync, status, config

**Features:**
```bash
idlergear wiki push              # Push references to Wiki
idlergear wiki pull              # Pull Wiki pages to references
idlergear wiki sync              # Bidirectional sync
idlergear wiki sync --conflict local   # Use local on conflicts
idlergear wiki sync --conflict remote  # Use remote on conflicts
idlergear wiki status            # Show sync status
idlergear wiki config            # Manage configuration
```

**Smart Conflict Resolution:**
- Manual (default) - List conflicts for user review
- Local - Always use IdlerGear version
- Remote - Always use Wiki version
- No data loss - Never overwrites without permission

**Configuration:**
```toml
[wiki]
enabled = true
auto_sync = true
sync_interval = 300  # seconds
```
